Prompt #28

background-image makes the background of the box an image.

background-position lets you change where the image is located in the background.

background-size changes how big or small the background is in relation to the box and / or image.

background-repeat lets you rule how you want the image to be repeated if there is enough space within the box for the image to do so, or if you want to make it so that the image does not repeat at all.

Changing the properties does not always yield a change, but it does add rules to the image and its box.
